How to use
  • Each food entry should be formatted like this:
    30g potatoes
  • You can enter calories manually. Try typing
    400kcal
  • Instead of typing the weight, you can type the number of items by adding an 'x':
    5x olives
  • You can set custom foods if you know the calories per 100g.
    Try this:
    15g chocolate 200kcal
